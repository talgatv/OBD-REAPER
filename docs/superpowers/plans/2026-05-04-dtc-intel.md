# DTC INTEL Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a built-in DTC code database to OBD-REAPER so every scan automatically decodes raw OBD2 bytes into human-readable descriptions with severity classification (CRITICAL/WARNING/INFO), shown in the live scan log and enriched in the final `.md` report.

**Architecture:** A new `core/dtc_db.py` module holds ~150 DTC codes plus three pure functions (`lookup`, `parse_dtcs`, `enrich_line`). `reporter.py` calls `enrich_line` per checkpoint line when writing the `.md` (the `.raw` keeps raw bytes). `tui.py` calls `enrich_line` in the right panel to color-code CRITICAL (red/bold) and WARNING (yellow) lines at display time.

**Tech Stack:** Python 3 stdlib only (`unittest`, no external deps). No pip. Curses for TUI.

```python
def lookup(code):
    """Return (description, severity) for a DTC code string, or (None, UNKNOWN)."""
    code = code.upper().strip()
    if code in DTC_DB:
        desc, sev = DTC_DB[code]
        return desc, sev
    return None, UNKNOWN


def parse_dtcs(raw):
    """Parse ELM327 response string into list of DTC code strings.

    '43 01 33 03 01' -> ['P0133', 'P0301']
    '43 00 00'       -> []
    'NO DATA'        -> []
    """
    parts = raw.upper().split()
    if parts and parts[0] in _DTC_PREFIXES:
        parts = parts[1:]
    dtcs = []
    for i in range(0, len(parts) - 1, 2):
        try:
            b1 = int(parts[i], 16)
            b2 = int(parts[i + 1], 16)
        except (ValueError, IndexError):
            break
        if b1 == 0 and b2 == 0:
            continue
        type_bits = (b1 >> 6) & 0x03
        d1 = (b1 >> 4) & 0x03
        d2 = b1 & 0x0F
        d3 = (b2 >> 4) & 0x0F
        d4 = b2 & 0x0F
        dtcs.append("{}{}{}{:X}{:X}".format(_TYPE[type_bits], d1, d2, d3, d4))
    return dtcs


def enrich_line(line):
    """Enrich a raw scan log line if it contains DTC response data.

    Returns (enriched_text, highest_severity).
    If no DTCs found, returns (line, None).
    """
    if "->" not in line:
        return line, None
    _, response = line.split("->", 1)
    parts = response.strip().upper().split()
    if not parts or parts[0] not in _DTC_PREFIXES:
        return line, None
    dtcs = parse_dtcs(response.strip())
    if not dtcs:
        return line, None
    decoded = []
    highest = INFO
    for code in dtcs:
        desc, sev = lookup(code)
        decoded.append((code, desc or "Unknown code", sev))
        if sev == CRITICAL:
            highest = CRITICAL
        elif sev == WARNING and highest != CRITICAL:
            highest = WARNING
    suffix = "".join(
        "\n  ► {:<8} {:<42} [{}]".format(c, d, s)
        for c, d, s in decoded
    )
    return line + suffix, highest
```
